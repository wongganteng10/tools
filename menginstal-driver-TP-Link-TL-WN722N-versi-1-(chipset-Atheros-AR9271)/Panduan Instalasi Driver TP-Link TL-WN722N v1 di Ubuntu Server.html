üß© 1. Cek Versi dan Chipset Adapter

Pastikan adapter kamu versi 1, karena hanya versi ini yang memakai chipset Atheros AR9271 dan bisa langsung menggunakan driver ath9k_htc bawaan Linux. 

lsusb

Contoh output untuk versi v1:

Bus 002 Device 004: ID 0cf3:9271 Atheros Communications, Inc. AR9271 802.11n

‚úÖ Jika muncul 0cf3:9271, berarti benar versi v1.
‚ùå Jika berbeda (misal 2357:010c), berarti versi v2/v3 ‚Äî butuh driver lain.

‚öôÔ∏è 2. Aktifkan Modul Driver Bawaan

Ubuntu Server biasanya sudah punya driver ath9k_htc. Aktifkan dan pastikan firmware tersedia:

sudo apt update
sudo apt install linux-firmware wireless-tools wpasupplicant net-tools
sudo modprobe ath9k_htc

Cek apakah terdeteksi:

iwconfig

Jika muncul:

wlan0     IEEE 802.11  ESSID:off/any  Mode:Managed  Access Point: Not-Associated

‚úÖ Driver sudah berfungsi.

üß∞ 3. Tambahkan Modul agar Aktif Otomatis Saat Boot

echo ath9k_htc | sudo tee -a /etc/modules

üåê 4. Hubungkan ke Wi-Fi (CLI)

Opsi 1 ‚Äî Menggunakan NetworkManager (nmcli)

sudo apt install network-manager
sudo systemctl enable NetworkManager
sudo systemctl start NetworkManager
nmcli device wifi list
nmcli device wifi connect "Nama_WiFi" password "KataSandiWiFi"
nmcli device status

Opsi 2 ‚Äî Menggunakan wpa_supplicant (lebih ringan)

sudo nano /etc/wpa_supplicant.conf

Isi dengan:

network={
    ssid="Nama_WiFi"
    psk="KataSandiWiFi"
}
sudo wpa_supplicant -B -i wlan0 -c /etc/wpa_supplicant.conf
sudo dhclient wlan0

üß™ 5. Cek Koneksi Internet

ping -c 4 google.com

Jika berhasil reply ‚Üí ‚úÖ koneksi sudah aktif.

ü©∫ 6. Jika Tidak Terdeteksi

sudo dmesg | grep ath9k_htc

Jika muncul:

firmware ath9k_htc/htc_9271.fw not found

Unduh manual firmware:

sudo mkdir -p /lib/firmware/ath9k_htc/
sudo wget https://github.com/qca/open-ath9k-htc-firmware/raw/master/htc_9271.fw -O /lib/firmware/ath9k_htc/htc_9271.fw
sudo modprobe -r ath9k_htc
sudo modprobe ath9k_htc

üí° Mengapa Bukan wlan0?

Ubuntu kini memakai sistem Predictable Network Interface Names, sehingga nama Wi-Fi menjadi wlx... bukan wlan0.

GRUB_CMDLINE_LINUX="net.ifnames=0 biosdevname=0"

Ubah GRUB, jalankan sudo update-grub, lalu reboot untuk mengembalikan nama menjadi wlan0.
